## Overview

<a href="#">CrossOver Mac</a> is a compatibility layer developed by CodeWeavers that allows macOS users to install and run Windows applications without purchasing a Windows license or setting up a virtual machine. Built on the open-source Wine project, CrossOver wraps Windows software in a translation environment that converts Windows API calls into macOS-native equivalents, enabling genuine Windows execution speed directly on your Mac hardware.

The <a href="#">bottle system</a> organizes Windows applications into isolated containers, each with its own virtual Windows environment, registry, and file system. This architecture prevents applications from interfering with each other and allows different programs to use different compatibility configurations. <a href="#">CrossOver's GUI</a> makes creating, managing, and removing bottles straightforward without requiring any command-line knowledge — the complexity of Wine configuration happens behind the scenes.

<a href="#">Application compatibility</a> spans thousands of Windows programs including productivity software, creative tools, utilities, and a large catalog of Windows-only games. CodeWeavers maintains an actively updated compatibility database at WineHQ where users can check compatibility ratings for specific applications before installing. <a href="#">The CrossTie system</a> provides one-click installation scripts for hundreds of officially supported applications, automating the configuration steps required for reliable operation.

The <a href="#">GPU acceleration</a> in CrossOver passes through Metal, Apple's graphics API, for games and graphics-intensive applications. DirectX translation via DXVK and D3DMetal converts Direct3D calls to Metal commands, enabling many Windows games to run with hardware-accelerated 3D graphics on Mac. This approach delivers playable frame rates for a wide range of titles without the overhead of a full virtual machine.

## Key Features

- <a href="#">Wine-based Windows compatibility</a> runs thousands of Windows apps and games directly on macOS without a license
- <a href="#">Isolated bottle containers</a> keep each Windows application in its own sandboxed environment for clean management
- <a href="#">One-click CrossTie installers</a> automate setup for hundreds of officially supported Windows programs
- <a href="#">DirectX to Metal translation</a> via DXVK enables GPU-accelerated 3D graphics in Windows games on Mac
- <a href="#">No rebooting required</a> — switch between macOS and Windows applications instantly in the same session
- <a href="#">Compatibility database</a> at WineHQ lists thousands of tested applications with community ratings
- <a href="#">macOS integration</a> places Windows apps in Launchpad and supports Dock pinning for native-feel access
- <a href="#">Apple Silicon support</a> runs Windows x86 applications on M1 M2 M3 and M4 chips via Rosetta translation
- <a href="#">Regular Wine updates</a> improve compatibility and performance with each CrossOver release cycle

## Additional Information

CrossOver represents a practical solution to the Windows-only software problem that Mac users have faced for decades. By funding Wine development and employing core Wine engineers, CodeWeavers accelerates improvements that benefit both CrossOver customers and the broader open-source community.

Updates ship regularly with each new CrossOver version, incorporating Wine improvements and compatibility fixes for both recent macOS releases and newly released Windows software. The development team prioritizes the most-requested applications and games for compatibility work each release cycle.
